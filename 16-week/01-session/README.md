# Sistema de Gestión de Inventario de Hogar

## Descripción
Sistema para gestionar y clasificar objetos físicos de una casa con información predefinida. El sistema contiene un inventario fijo de objetos organizados por categorías.

## Características
- **Herencia**: Clase base abstracta `ObjetoHogar` y clases derivadas específicas
- **Polimorfismo**: Métodos que se comportan diferente según el tipo de objeto
- **Clases Abstractas**: `ObjetoHogar` define la interfaz común
- **Composición**: `Habitacion` compuesta por múltiples objetos
- **Manejo de Errores**: Validaciones y excepciones personalizadas
- **Colecciones**: Uso de listas, diccionarios y enumeraciones
- **Archivos**: Lectura/escritura de inventario en JSON
- **Buenas Prácticas**: Tipado, documentación, encapsulación
